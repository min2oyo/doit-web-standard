2. 스타일

스타일 형식

    p {
        text-align: center;
        color: blue;
    }

    선택자 {
        속성1: 속성값1;
        속성2: 속성값2;
    }

스타일 시트
- 브라우저 기본 스타일

- 사용자 스타일
    - 인라인 스타일

        <태그 style="속성: 속성값;">

    - 내부 스타일 시트

        <style>
            선택자 {
                속성: 속성값;
            }
        </style>

    - 외부 스타일 시트

        <link rel="stylesheet" href="외부 스타일 시트 파일 경로">

3. 선태자

전체 선택자

    * {margin: 0;}

    * {속성: 값;}

    - universal selector
    - 문서의 모든 요소에 적용
    - 웹 브라우저의 기본 스타일을 초기화할 때 자주 사용
        - 여백(마진, 패딩) 조절

타입 선택자

    p {font-style: italic;}

    태그명 {스타일 규칙;}

    - type selector
    - 특정 태그를 사용한 모든 요소에 적용

클래스 선택자

    .bg {background-color: #ddd;}

    .클래스명 {스타일 규칙;}

    - class selector
    - 문서에 여러번 적용할 수 있음

id 선택자

    #container {width: 500px;}

    #아이디명 {스타일 규칙;}

    - id selector
    - 문서에 한 번만 적용할 수 있음
    - 주로 문서의 레이아웃과 관련된 스타일을 지정함
    - 주로 자바스크립트로 요소를 구별할 때 사용

그룹 선택자

    h1, h2 {text-align: center;}

    선택자1, 선택자2 {스타일 규칙}

    여러 요소에 같은 스타일을 적용할 때 사용

4. 캐스케이딩
- Cascading Style Sheet
- 스타일시트에서 우선순위가 위에서 아래 계단식으로 적용된다는 의미
- 스타일 우선순위: 스타일 규칙의 중요도와 적용 범위에 따라 우선순위가 결정되고 그 우선순위에 따라 위에서 아래로 스타일을 적용함
- 스타일 상속: 태그의 포함 관계에 따라 부모 요소의 스타일을 자식 요소로 위에서 아래로 전달

스타일 우선순위
- 중요도

    사용자 스타일 > 제작자 스타일 > 브라우저 기본 스타일

    - 사용자 스타일: 고대비

- 적용 범위

    !important > 인라인 스타일 > id 스타일 > 클래스 스타일 > 타입 스타일

- 작성 순서

    나중에 작성한 스타일이 적용됨

스타일 상속
- 배경색과 배경 이미지는 스타일 상속이 되지 않음
